使用Github+Hexo搭建一个博客

  • Github
  • Hexo

今天教给大家的是一个使用Github的Pages功能来搭建一个属于自己的个人博客，博客的模板我选用的是HEXO，主题选择MOxFIVE开发的yelee模板。
1.安装需要使用的工具git,npm,hexo,nodejs
2.新建github存储库
3.生成连接github的公私钥
4.上传公钥到github上
5.测试github连接
6.安装hexo,生成博客
7.发布hexo博客

一、安装工具

  • 在搭建HEXO站点之前我们需要准备一些东西，在系统上安装好git，npm，nodejs，安装步骤如下：

1.安装git

  • 安装git非常的简单，我的系统是kali-rolling，而kali是基于debian的一个发行版本，所以我们安装git工具可以直接使用apt工具就可以了。

2.安装nodejs,npm

  • 在我的系统上默认是安装了nondejs的，但是并没有npm这个工具，而npm这个工具在之后安装hexo网站的时候是需要使用到的，所以我们可以去网站上下载nodejs的安装脚本行安装。

二、新建github存储库

1.新建存储库

  • 我们登录github之后，通过右上角的+号添加一个新的存储库。

2.存储库的名称为自己的用户名加上github.io

三、生成连接github的公私钥

1.建立完存储库后添加ssh密钥到github上方便hexo发布

2.生成新的ssh公私密钥对

3.将以.pub结尾的公钥内容复制到github上

4.查看github上的密钥信息

四、使用ssh连接github

1.测试密钥上传是否成功

  • 当我们第一次使用ssh连接github的时候，会需要确认是否连接，写yes就可以了

2.确认连接之后设置下用户信息，分别是用户名和E-mail。

3.下载新建的github存储库

五、安装hexo，并发布博客

1.安装hexo非常的简单，只需要使用npm来安装。

2.下载安装hexo博客。

3.生成博客

4.发布博客到github上

1.发布博客需要安装hexo插件
2.配置blog的配置文件_config.yml来支持github上传
3.发布博客
